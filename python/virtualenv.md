# Virtualenvについて

参考  
https://python.keicode.com/devenv/virtualenv.php

仮想の開発環境

pipで入れる

「作る→入る→抜ける　」だけっぽい

「virtualenv 環境名」すると、フォルダができる

    drwxr-xr-x 2 root root 305  2月 28 12:37 bin
    drwxr-xr-x 3 root root  23  2月 28 12:37 lib
    -rw-r--r-- 1 root root 296  2月 28 12:37 pyvenv.cfg
    
## 環境に入る

source 環境名/bin/activate

## 環境から抜ける

deactivate

非常に簡単
